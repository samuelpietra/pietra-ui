## Getting Started

### Installation

```bash
npm install pietra-ui react react-dom
```

### Setup

Wrap your app with `ThemeProvider` and import the stylesheet:

```tsx
import { ThemeProvider, Button, Text } from "pietra-ui";
import "pietra-ui/styles.css";

function App() {
  return (
    <ThemeProvider>
      <Text size="5" weight="bold">Hello Pietra</Text>
      <Button>Click me</Button>
    </ThemeProvider>
  );
}
```

## Highlights

- **Compound components** — compose declaratively, just like Radix itself
- **Virtualized views** — built-in for table, list, and grid layouts handling large datasets
- **Fully typed** — strict TypeScript with exported types for every component
- **Themed out of the box** — dark mode, accent colors, and radius tokens just work via Radix Themes
- **Accessible** — inherits Radix's WAI-ARIA patterns
